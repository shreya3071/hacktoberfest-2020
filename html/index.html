Rules for Hacktoberfest-2020

  1. To earn your Hacktoberfest tee or tree reward, you must need to register and make four valid pull requests (PRs) between October 1-31 (in any time zone).
  2. PRs can be made to any public repository on GitHub, not only the ones with issues labeled Hacktoberfest.
  3. If a maintainer reports your pull request as spam or behavior not in line with the project’s code of conduct, you will be ineligible to participate.
  4. This year, the first 75,000 participants who successfully complete the challenge will be eligible to receive a prize.
